## Installation

<CodeBlockCommand command="headcn@latest add textarea" />

## Usage

```tsx
import { Textarea } from "@/components/ui/textarea"
```

```tsx
<Textarea />
```

You can pass any props to an `Textarea` that you'd normally pass to the native `textarea` element.
