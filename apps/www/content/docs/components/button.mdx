## Installation

<Command _command="headcn@latest add button" />

## Usage

```tsx
import { Button } from "@/components/ui/button"
```

```tsx
<Button>Star me!</Button>
```

You can pass any props to a `Button` that you'd normally pass to the native `button` element.
